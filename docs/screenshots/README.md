# Screenshots

Add screenshots of your application here to make your README more attractive.

## Recommended Screenshots

1. **Dashboard View** - Main UI showing accounts and orders
2. **Create Order Form** - Order creation interface
3. **Orders Table** - List of executed orders
4. **Execution Logs** - Audit trail view
5. **H2 Console** - Database view (optional)

## How to Add Screenshots

1. Take screenshots of your running application
2. Save them in this folder with descriptive names:
   - `dashboard.png`
   - `create-order.png`
   - `orders-table.png`
   - `execution-logs.png`

3. Reference them in README.md:
```markdown
## Screenshots

### Dashboard
![Dashboard](docs/screenshots/dashboard.png)

### Create Order
![Create Order](docs/screenshots/create-order.png)
```

This makes your repository much more attractive to recruiters!
